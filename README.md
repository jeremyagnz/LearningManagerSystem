# Sistema de Gestión de Aprendizaje

## Descripción Corta
Un sistema diseñado para gestionar cursos y aprendizaje de manera eficiente.

## Características
- Gestión de cursos
- Seguimiento de progreso
- Informes y análisis

## Requisitos
- Python 3.8+
- Django 3.1+
- Base de datos (SQLite, PostgreSQL, etc.)

## Instalación
1. Clona el repositorio: `git clone <url del repositorio>`
2. Instala las dependencias: `pip install -r requirements.txt`

## Configuración (Ejemplo de env)
```
DEBUG=True
DATABASE_URL=sqlite:///db.sqlite3
SECRET_KEY=tusecretkey
```

## Ejecución
Ejecuta `python manage.py runserver` para iniciar el servidor.

## Estructura del Proyecto
- `manage.py`
- `app/` - Contiene la lógica de la aplicación
